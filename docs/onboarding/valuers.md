# Valuer Onboarding Pack

This guide is for valuers and surveyors responsible for turning case instructions into evidence-backed valuation conclusions.

## First-Day Setup

- confirm web login works
- confirm mobile login works if inspections are part of the role
- open:
  - cases
  - case detail
  - analysis workbench
  - reports
  - comparables
- review notification behavior for case assignment and report review

## First-Week Learning Path

### Day 1

- read the [valuer SOP](/Users/michaeladeleye/Documents/product/ERP Surveyors/docs/sops/valuers.md)
- review one seeded case from intake to final issue
- understand the case stage order

### Day 2

- open a seeded case with a submitted inspection
- inspect the document, client, property, and activity context
- review attached comparables

### Day 3

- attach or create comparables
- edit relevance, adjustment, and weighting
- complete assumptions and commentary in the workbench

### Day 4

- generate a draft report
- inspect how the analysis and commentary appear in the report
- submit a report for review

### Day 5

- work through review comments on a report
- update the analysis/report and resolve comments correctly

## Role Readiness Checklist

The team lead should confirm the person can:

- interpret case context properly
- attach and evaluate comparables correctly
- complete the analysis workbench properly
- explain the difference between weighted indication and concluded value
- generate and submit a draft report
- respond properly to review comments

## Daily Habits

- review assigned cases and priorities first thing
- move overdue and review-dependent items first
- keep analysis commentary clear and decision-oriented
- avoid submitting reports with unresolved internal gaps

## Weekly Habits

- review evidence quality across active cases
- review rejected or comment-heavy reports for pattern learning
- review turnaround on assigned work

## KPIs To Watch

- cases completed per period
- average time from inspection submitted to analysis completed
- average time from analysis complete to report submitted
- reviewer rejection rate
- blocking comment rate per submitted report
- overdue assigned case count

## 30-Day Signoff Questions

- can the person independently produce a review-ready report?
- can they use comparables and adjustments credibly?
- are they documenting assumptions and reconciliation clearly?
- do they respond well to reviewer feedback?
